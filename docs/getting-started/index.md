# Getting Started

This section helps you get up and running with the Gnosis Analytics platform. Whether you want to start making API calls immediately or understand the full platform architecture first, choose the path that fits your needs.

## Choose Your Path

### I want to use the API

Head to the **[Quick Start](quickstart.md)** guide. You will make your first API call in under a minute -- no API key required for public endpoints.

### I want to understand the platform

Start with the **[Platform Overview](platform-overview.md)** to learn about the 13-repo ecosystem, then explore the **[Architecture](architecture.md)** page for a detailed technical breakdown of the 4-layer data pipeline.
